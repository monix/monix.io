trait Streamable[F[_]] extends Suspendable[F] with Memoizable[F] with MonadError[F, Throwable] with MonadPlus[F] with CoflatMap[F]

Groups common type-classes for things that represent (possibly asynchronous) streams (e.g. Observable).
